# 🎸 Guitar Fretboard Diagram Maker

A premium, interactive web application designed for guitarists, teachers, and students to create, save, and export high-quality guitar fretboard diagrams.

## ✨ Features

- **Interactive Fretboard**: Add marks (circles, squares, triangles, stars, and more) to any string or fret with a single click.
- **Dynamic Customization**:
  - Adjust the number of **strings** (4 to 8) and **frets** (up to 24).
  - Change the **Start Fret** to visualize patterns anywhere on the neck.
  - Choose from a curated palette of **vibrant colors** for your marks.
- **Save & Load System**:
  - **Quick Save**: Save your creations instantly to local storage.
  - **Collapsible Sidebar**: Browse your library of saved diagrams in a sleek, right-side panel without leaving your drawing.
- **High-Quality Export**: Download your diagrams as **crystal-clear PNG images** with a single click, perfect for sharing or printing.
- **Note Labeling**: Right-click any mark to add custom text (fingering, intervals, or note names).
- **Responsive & Premium Design**: Enjoy a dark-themed, glassmorphic interface with smooth animations and a "wood-finish" fretboard aesthetic.

## 🛠️ Tech Stack

- **React**: For a reactive and modern UI components.
- **Framer Motion**: For silky-smooth animations and transitions.
- **Lucide React**: For crisp, consistent iconography.
- **HTML-to-Image**: High-fidelity SVG to PNG conversion.
- **Vite**: Ultra-fast development and build pipeline.

## 🚀 Getting Started

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/hankyukim89/Guitarfretboard.git
    ```
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Run Locally**:
    ```bash
    npm run dev
    ```
